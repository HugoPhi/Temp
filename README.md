# MachineLearningLabs

[中文版本](#中文版本)

This repository contains implementations of machine learning models from scratch using numpy. It is designed for educational purposes, to help understand the inner workings of various machine learning algorithms. 

## Features
- Implementations of various machine learning models.
- Pure Python (numpy-based) implementations for learning purposes.
- MIT License.

## Requirements
- Python 3.x
- numpy

## How to use
1. Clone the repository:
    ```bash
    git clone https://github.com/HugoPhi/MachineLearningLabs.git
    ```
2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3. Run the tests:
    ```bash
    python -m unittest discover
    ```
