# 中文版本

[English Version](README.md)

本仓库包含基于 numpy 的机器学习模型从零实现，旨在帮助学习者理解各类机器学习算法的内部工作原理。

## 功能
- 多种机器学习模型的实现。
- 纯 Python (基于 numpy) 实现，适合学习用途。
- MIT 许可。

## 依赖要求
- Python 3.x
- numpy

## 使用方法
1. 克隆仓库：
    ```bash
    git clone https://github.com/HugoPhi/MachineLearningLabs.git
    ```
2. 安装依赖：
    ```bash
    pip install -r requirements.txt
    ```
3. 运行测试：
    ```bash
    python -m unittest discover
    ```
